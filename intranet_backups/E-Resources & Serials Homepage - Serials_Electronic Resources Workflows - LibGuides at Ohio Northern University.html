E-Resources & Serials Overview

Information listed on the Electronic Resources and Serials pages pertains to the processes of acquiring, adding, and making these resources accessible to our library users. This involves managing subscriptions and adding newly acquired databases to the library's shared catalog, discovery system, and LibGuides. These processes and their documentation are managed by the Electronic Resources Librarian.

As these processes contain some overlap between the Systems and Cataloging departments, some documentation housed in the guide below may be shared between the three.

  • E-Resources & Serials
    This guide contains information pertaining to the administration of electronic resources and serials acquired and available through Heterick Memorial Library, as managed by the Electronic Resources Librarian.

Cards

Closing Cards

  1. Go to Summary to make sure Law does not have:
    • If Law has holdings, mark for Librarian, and do nothing else with the card.
  2. Go to View and copy down OCLC number:
    • If all items are weeded, put –W after number for Catalog Librarian.
  3. Click Summary to return to record.
  4. Check card and write down last issue received
    • If there are gaps, make note and leave for Librarian. Do not do anything else.
  5. Go to Record
    • Put end date in Lib. Has
      • Use following format (no spaces): v.91:no.1(2016)
      • Leave 3 spaces between date and ***Gaps
      • If last issue received is the end of a volume, just enter the volume: v.90(2015).
    • Next to SCODE 1, double click and select X for inactive serial.
    • Save
  6. ​Go back to card
    • Right click on any box
    • Select card menu -> delete card
    • Save
  7. Go to Summary and verify that status is No Card

Librarian: Anything completely weeded needs to be removed from FTF.

Deleting Record (LIBRARIAN)

If we’re weeding all, there are no law holdings and no order record:

  • Send OCLC number.
  • Edit bib record, then go to File->delete bib record

If there is an order record:

  • Send the OCLC number so cataloger can remove the holdings
  • Delete any attached checkin or item records
  • Go to the bib record and in the SUPPRESS box at the top right, change the dash (-) for DIS LOC&CENT to an (s) for SUPLOC/NOCENT.

(Explanation: When an order record is present, even if we aren't keeping any issues, we can't get rid of the bib record until Kathleen deletes the order record. We want to suppress the whole thing from view if there are no bound issues on the second floor for anyone to use.)

Law has holdings, we’re not keeping any issues (LIBRARIAN):

  • Send OCLC
  • Summary->select card (starts w/ C) and item record (starts w/ I) [MAKE SURE IT IS OURS!]
  • Edit record, go to location -> multi. Double click and remove HML location.
  • Go to order record and suppress under Code 4

Keeping bound issues:

  • Send OCLC number so cataloger can verify Local Holdings Record
  • Update holdings field to reflect the last issue we ever received and kept
  • Nothing is deleted or suppressed
  • Closing cards with screen shots

Updating Dates on Closed Cards

  • Click on Record in left column
  • Edit Lib Has field, using format v.1(1961)-v.56:no.2(2017) (if there are no gaps in holdings)
    • If there are gaps: add them using the following format (including the asterisks) after the dates ***Gaps: v.106;v.108:no.8*** 
  • When finished, click save
  • Instructions with screen shots

Kanopy

Kanopy Request Process

  1. Electronic Resource Librarian receives request from Kanopy
  2. ERL checks catalog to determine we don't already have as a DVD
  3. ERL contacts requester.:
    • Student requester and we have on DVD: tell student DVD is available and there is a player in the library. We will not license streaming in this situation.
    • Student requester, do not have on DVD: verify need urgency and format necessity. Get on DVD if possible.
    • Faculty requester and we have on DVD: see if DVD is acceptable for their needs
    • Faculty requester and do NOT have on DVD: see how soon the film is needed and if they plan to watch in class or outside of class. If not urgent and watching in class, order DVD. 
  4. If streaming required/need within 2 weeks, ERL completes order on Kanopy (invoice instructions below)
  5. If streaming needed but not immediate, complete order on Kanopy with delayed start (invoice instructions below)
  6. In comments, request that Kanopy takes payment from funds on deposit

Kanopy Invoice Process

  1. Invoices are sent to ERL on the last day of the month on a monthly basis.
  2. Forward copy to director with updated balance from Kanopy dashboard (under packages)
  3. Update Kanopy Triggers spreadsheet (in Databases->Kanopy folder)
  4. Forward invoice email with updated spreadsheet to Cataloging Librarian

Note: CL adds item records to Sierra for purchases. You can look them up in Sierra by doing a barcode search for the word Kanopy. Updated quarterly with this list.

For mediated purchases, also include Acquisitions Asst.
